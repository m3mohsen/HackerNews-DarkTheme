Hacker News - Dark Theme

A chrome extension to install a dark theme for hacker news.

Installation

  1. Download the ext directory from github or just type:
$ git clone https://github.com/m3mohsen/HackerNews-DarkTheme.git
  1. Go to your browser and type: chrome://extensions/
  2. Enable the Developer mode on the top
  3. Load the ext directory by clicking on "Load unpacked extension" button
  4. If you see the Hacker News icon, you're good to go!
